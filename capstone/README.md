# Capstone Project

## Overview
This project is a capstone application that serves as a demonstration of skills and knowledge acquired throughout the course. It includes a main application file, utility functions, and test cases to ensure functionality.

## Project Structure
```
capstone
├── src
│   ├── index.js          # Entry point of the application
│   └── utils
│       └── helpers.js    # Utility functions
├── tests
│   └── index.test.js     # Test cases for the application
├── package.json          # NPM configuration file
└── README.md             # Project documentation
```

## Installation
To install the necessary dependencies, run the following command:

```
npm install
```

## Usage
To start the application, use the following command:

```
npm start
```

## Running Tests
To run the test cases, execute:

```
npm test
```
